### Features

reLift-HTML aims to be simple, easy to use and helps you do much more. 

- Very small
- Web Components
- Custom Elements
- Template literals
- Directives
- Data binding
- One way data flow
- Two-Way Data Binding on forms
- Computed properties
- Event Handling
- Lifecycle
- State management
- No JSX 
- No dependency
- No virtual DOM
- No need for CLI
- No build, the real DOM does it!
